Differences Between var, let and const:
  var is used in a concept, let is used in a block and const is used in also a block but we can not reassign const after first declaration.
  
About Spread Operator:
 This operator expands arrays or object into individual elements so that we can easily use it for copying,merging or passing multiple values.

 Difference between map(), filter() and forEach():
    map function returns a new array, filter function returns filtered elements and foreach function returns every element from an array or object.

Arrow function:
  This function is a shorter syntax for writing a function. It indicates to the scope of the function.

 Template literates:
 When strings are written using backticks with allowing variables with ${} are called template literates. This supports multi-line strings easily.
